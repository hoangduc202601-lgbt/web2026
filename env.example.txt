# 1. Tạo Integration:
#    - Truy cập: https://www.notion.so/my-integrations
#    - Click "New integration"
#    - Đặt tên và chọn workspace
#    - Copy "Internal Integration Token" -> NOTION_API_KEY

# 2. Chia sẻ Database với Integration:
#    - Mở database trong Notion
#    - Click "..." ở góc phải -> "Add connections"
#    - Chọn integration vừa tạo

# 3. Lấy Database ID:
#    - Mở database trong Notion
#    - Copy ID từ URL (32 ký tự sau tên workspace)

# 4. Cấu trúc Database Articles đề xuất:
#    - Title (title): Tiêu đề bài viết
#    - Slug (rich_text): URL thân thiện
#    - Excerpt (rich_text): Mô tả ngắn
#    - Category (select): Danh mục
#    - Tags (multi_select): Thẻ
#    - Author (rich_text): Tác giả
#    - Status (select): Published / Draft
#    - Featured (checkbox): Nổi bật
#    - Views (number): Lượt xem
#    - PublishedAt (date): Ngày đăng
